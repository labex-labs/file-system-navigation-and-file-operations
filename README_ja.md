この包括的なコースで、必須の Linux ファイルシステム操作を習得しましょう。実践的なラボと魅力的な課題を通じて、ディレクトリ構造のナビゲーション、ファイルのリスト表示、ファイルの作成と管理、ファイルのコピーと移動、および効率的なファイル操作のためのワイルドカードの使用方法を学びます。Linux ファイル管理に習熟したいすべての人に最適です。
